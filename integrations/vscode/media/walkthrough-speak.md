## 🔊 Hear it

```
say-it kubectl   →  "koob control"     (Kelsey Hightower, KubeCon)
say-it nginx     →  "engine X"          (nginx.org official)
say-it GIF       →  "jif"               (Wilhite, NYT 2013)
say-it YAML      →  "yam ul"            (yaml.org FAQ)
say-it Ghostty   →  "ghost ee"          (Mitchell Hashimoto)
```

The CLI is canonical (macOS `say`). The VS Code extension uses the same dictionary.
