How to pronounce Postgres vs PostgreSQL

Same database. Postgres ("POST-gress") is the short colloquial name; PostgreSQL ("POST-gress-Q-L" or "POST-gress sequel") is the official name documented on the project site. Both readings of PostgreSQL appear in the wild — the official FAQ accepts "post-gres-Q-L".

Postgres /ˈpoʊstɡrɛs/ — post gress
PostgreSQL /ˈpoʊstɡrɛs kjuː ˈɛl/ — post gress Q L
