cartesia 怎么读

产品 社区共识

kar tee zhuh /kɑrˈtiʒə/ mp3

cartesia 在英语中的标准读法是 「kar tee zhuh」（/kɑrˈtiʒə/）。

Real-time voice-AI lab (Stanford 2023; Karan Goel, Albert Gu) behind the Sonic TTS models, built on state-space models (SSMs); rising ElevenLabs rival. Name derives from "Cartesian" (math). STRESS: kar-TEE-zhuh — soft "zh" as in "vision". Cartesia's own docs give IPA kʰ ɑ ɹ tʲ i ʒ ɐ, formed by taking "cartesian" and replacing the final "-an" with an "uh" sound.

其他读法：

  • 「kar tee zee uh」 /kɑrˈtiziə/

来源：Cartesia Docs — Custom Pronunciations
