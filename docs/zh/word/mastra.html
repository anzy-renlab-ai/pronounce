mastra 怎么读

项目 社区共识

mass truh /ˈmæstrə/ mp3

mastra 在英语中的标准读法是 「mass truh」（/ˈmæstrə/）。

Open-source TypeScript AI agent framework from the ex-Gatsby team (YC-backed, 1.0 in early 2026). Coined name; etymology undocumented. Dominant English reading is MASS-truh — short A as in 'master', stress on the first syllable. A spelling-pronunciation MAH-struh is sometimes heard; no documented creator FAQ resolves it, hence community-consensus. STRESS: MASS-truh.

其他读法：

  • 「mah struh」 /ˈmɑːstrə/

来源：Mastra GitHub README (mastra-ai/mastra)
