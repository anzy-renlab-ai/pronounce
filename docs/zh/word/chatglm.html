ChatGLM 怎么读

产品 社区共识

chat jee ell em /ˌtʃæt dʒiː ɛl ˈɛm/ mp3

ChatGLM 在英语中的标准读法是 「chat jee ell em」（/ˌtʃæt dʒiː ɛl ˈɛm/）。

Zhipu AI / Tsinghua KEG open bilingual model series. Name = "Chat" + GLM, where GLM is an initialism for "General Language Model" spoken letter-by-letter: chat-G-L-M, NOT read as a word "glim". STRESS: chat-jee-ell-EM (final letter takes the beat). Bare "GLM" is already in the dict; this is the searched product name.

来源：THUDM/ChatGLM-6B GitHub README

读对项目名只是工程师文化的一部分 — 但开会时被人轻轻纠正、或者在 podcast 上被嘉宾礼貌打断的滋味，并不好受。Pronounce 是一个开源的开发者词汇发音词典，每条都标注来源、信心等级，并且有预渲染的音频可以直接听。
