youki 怎么读

工具 创作者已明确

yoh kee /ˈjoʊki/ mp3

youki 在英语中的标准读法是 「yoh kee」（/ˈjoʊki/）。

Container runtime implementing the OCI runtime-spec in Rust, a runc alternative. The official README and docs state it is pronounced /joʊki/ or "yoh-key", after the Japanese word 容器 (yōki) meaning "a container". The "ou" is a Japanese long o, so it is not "YOO-kee".

来源：youki README — About the name ("youki is pronounced as /joʊki/ or yoh-key")
