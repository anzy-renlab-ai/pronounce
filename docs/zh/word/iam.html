I A M /ˌaɪ eɪ ˈɛm/ mp3

IAM 在英语中的标准读法是 「I A M」（/ˌaɪ eɪ ˈɛm/）。

Letters "I-A-M". Identity and Access Management.

来源：AWS IAM

读对项目名只是工程师文化的一部分 — 但开会时被人轻轻纠正、或者在 podcast 上被嘉宾礼貌打断的滋味，并不好受。Pronounce 是一个开源的开发者词汇发音词典，每条都标注来源、信心等级，并且有预渲染的音频可以直接听。

不用查第二次 — 在编辑器里直接听

装上 Pronounce 扩展，在编辑器里悬停 IAM（或任意技术词）就能听到读音，附 IPA 和来源，无需离开代码。中文环境下界面自动切中文。

安装到 VS Code Open VSX 也支持 Cursor · Windsurf · VSCodium · Zed

在命令行里听

$ say-it IAM # 主读音 × 3 次

安装 CLI：git clone https://github.com/anzy-renlab-ai/pronounce.git && cd pronounce && ./install.sh
