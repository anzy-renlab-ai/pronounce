maturin 怎么读

命令行工具 社区共识

mat uh rin /ˈmætʊrɪn/ mp3

maturin 在英语中的标准读法是 「mat uh rin」（/ˈmætʊrɪn/）。

STRESS first syllable: MAT-uh-rin (short 'a' as in 'mat', NOT ma-TUR-in). PyO3's official CLI to build & publish Rust-based Python wheels (used by polars, pydantic-core, etc.; pip-installable). The README doesn't document a pronunciation, but 'Maturin' is a real French Huguenot given name (cf. Stephen Maturin in O'Brian's Aubrey–Maturin novels), and references consistently put the stress on the FIRST syllable. A yod variant 'MAT-yuh-rin' also occurs.

其他读法：

  • 「mat yuh rin」 /ˈmætjʊrɪn/

来源：PyO3/maturin — official GitHub repo (formerly pyo3-pack)
