deepgemm 怎么读

项目 存在争议

deep jem /ˈdiːp ˌdʒɛm/ mp3

deepgemm 在英语中的标准读法是 「deep jem」（/ˈdiːp ˌdʒɛm/）。

DeepSeek's FP8/FP4 GEMM kernel library for Hopper/Blackwell, used as a MoE backend by vLLM/SGLang. "Deep" + "GEMM" (General Matrix Multiply). Pronunciation is genuinely split: most HPC/BLAS practitioners say GEMM as a single word rhyming with "gem"/"hem" (mirroring how S-GEMM = "ess-jem", D-GEMM = "dee-jem" are spoken), giving DEEP-jem. Others spell it out, deep-G-E-M-M. README gives no phonetic guidance. STRESS: DEEP-jem.

其他读法：

  • 「deep jee ee em em」 /ˌdiːp ˌdʒiː iː ɛm ˈɛm/

来源：DeepGEMM GitHub repository (deepseek-ai/DeepGEMM)
