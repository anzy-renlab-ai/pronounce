How to pronounce GUID

acronym contested

gwid /ɡwɪd/ mp3
or: G U I D /ˌdʒiː juː aɪ ˈdiː/

GUID is most commonly pronounced "gwid" (/ɡwɪd/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

Microsoft "gwid"; non-MS world tends letter-by-letter.

Alternate readings you might hear:

  • "G U I D" /ˌdʒiː juː aɪ ˈdiː/

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it GUID # primary × 3 + audible "or: …" for each alternate
$ say-it --alt GUID # focus on the first alternate
$ say-it --all GUID # primary AND every alternate, each repeated
$ say-it --solo GUID # primary only, no "or: …" tail
$ say-it --why GUID # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/pronounce.git && cd pronounce && ./install.sh
