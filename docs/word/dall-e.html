How to pronounce DALL-E

product community-consensus

dolly /ˈdɒli/
or: D A L L E /ˌdiː eɪ ɛl ˈiː/

DALL-E is most commonly pronounced "dolly" (/ˈdɒli/). This is the widely-used reading among engineers, though edge cases exist.

"DOLL-ee" (portmanteau of Dalí + WALL-E); some say it letter-by-letter.

Alternate readings you might hear:

  • "D A L L E" /ˌdiː eɪ ɛl ˈiː/

Source: OpenAI DALL-E

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it DALL-E # primary × 3 + audible "or: …" for each alternate
$ say-it --alt DALL-E # focus on the first alternate
$ say-it --all DALL-E # primary AND every alternate, each repeated
$ say-it --solo DALL-E # primary only, no "or: …" tail
$ say-it --why DALL-E # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/say-it.git && cd say-it && ./install.sh
