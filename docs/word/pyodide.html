How to pronounce Pyodide

product creator-clarified

pie uh dyed /ˈpaɪəˌdaɪd/ mp3

Pyodide is most commonly pronounced "pie uh dyed" (/ˈpaɪəˌdaɪd/). This pronunciation is documented by the project itself (see Pyodide GitHub issue #1631).

CPython compiled to WebAssembly for running Python in the browser; maintainers rth and hoodmane gave the IPA ˈpɑɪ.ə.ˌdɑɪd (Py + Iodide — rhymes with iodide).

Source: Pyodide GitHub issue #1631
