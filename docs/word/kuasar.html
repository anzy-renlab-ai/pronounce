How to pronounce Kuasar

project community-consensus

kway zar /ˈkweɪzɑr/ mp3

Kuasar is most commonly pronounced "kway zar" (/ˈkweɪzɑr/). This is the widely-used reading among engineers, though edge cases exist.

Kuasar is a CNCF Sandbox multi-sandbox container runtime written in Rust (Huawei Cloud, WasmEdge, openEuler) that unifies MicroVM, Wasm and app-kernel sandboxes behind one sandbox API. The name is simply "quasar" respelled with a K for Kubernetes — the maintainers' own CNCF application glosses it as "Kuasar(Quasar in Kubernetes)" — so it is said KWAY-zar, exactly like the astronomical object. Cold readers routinely mangle it as "koo-AH-sar" by reading the ua literally.

Source: CNCF Sandbox application #56 — maintainers' project summary: "Kuasar(Quasar in Kubernetes) is a low-level container runtime"
