How to pronounce etcd

product creator-clarified

et C D /ˌɛt siː ˈdiː/
or: et cetera D /ˌɛt ˈsɛtərə diː/

etcd is most commonly pronounced "et C D" (/ˌɛt siː ˈdiː/). This pronunciation is documented by the project itself (see etcd FAQ).

Project FAQ documents "et-cetera-distributed".

Alternate readings you might hear:

  • "et cetera D" /ˌɛt ˈsɛtərə diː/

Source: etcd FAQ

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it etcd # primary × 3 + audible "or: …" for each alternate
$ say-it --alt etcd # focus on the first alternate
$ say-it --all etcd # primary AND every alternate, each repeated
$ say-it --solo etcd # primary only, no "or: …" tail
$ say-it --why etcd # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/say-it.git && cd say-it && ./install.sh
