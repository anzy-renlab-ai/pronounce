How to pronounce LaTeX

tool creator-clarified

lay tek /ˈleɪtɛk/
or: la tek /ˈlɑːtɛx/

LaTeX is most commonly pronounced "lay tek" (/ˈleɪtɛk/). This pronunciation is documented by the project itself (see LaTeX project).

Lamport: "lay-tek" (or "lah-tek"); never "lay-teks". Documented on the about page.

Alternate readings you might hear:

  • "la tek" /ˈlɑːtɛx/

Source: LaTeX project

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it LaTeX # primary × 3 + audible "or: …" for each alternate
$ say-it --alt LaTeX # focus on the first alternate
$ say-it --all LaTeX # primary AND every alternate, each repeated
$ say-it --solo LaTeX # primary only, no "or: …" tail
$ say-it --why LaTeX # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/say-it.git && cd say-it && ./install.sh
