How to pronounce LaTeX

tool creator-clarified

lay tek /ˈleɪtɛk/ mp3

LaTeX is most commonly pronounced "lay tek" (/ˈleɪtɛk/). This pronunciation is documented by the project itself (see LaTeX project).

Lamport: "lay-tek" (or "lah-tek"); never "lay-teks". Documented on the about page.

Source: LaTeX project

How do you pronounce LaTeX?

LaTeX is pronounced "lay tek" (/ˈleɪtɛk/). Lamport: "lay-tek" (or "lah-tek"); never "lay-teks". Documented on the about page. Source: LaTeX project.

What is the IPA for LaTeX?

The IPA for LaTeX is /ˈleɪtɛk/, respelled "lay tek".
