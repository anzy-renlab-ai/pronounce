How to pronounce FLTK

tool creator-clarified

full tick /ˈfʊltɪk/ mp3

FLTK is most commonly pronounced "full tick" (/ˈfʊltɪk/). This pronunciation is documented by the project itself (see FLTK documentation — Introduction).

Veteran cross-platform C++ GUI toolkit; the official docs open with 'The Fast Light Tool Kit ("FLTK", pronounced "fulltick")'.

Source: FLTK documentation — Introduction
