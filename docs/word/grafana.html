How to pronounce Grafana

product community-consensus

gra fah na /ɡrəˈfɑːnə/

Grafana is most commonly pronounced "gra fah na" (/ɡrəˈfɑːnə/). This is the widely-used reading among engineers, though edge cases exist.

"gra-FAH-na" is the most common community reading.

Source: Grafana site

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it Grafana # primary × 3 + audible "or: …" for each alternate
$ say-it --why Grafana # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/say-it.git && cd say-it && ./install.sh
