How to pronounce Mockito

tool contested

mah kee toh /mɑːˈkiːtoʊ/ mp3
or: moh kee toh /moʊˈkiːtoʊ/

Mockito is most commonly pronounced "mah kee toh" (/mɑːˈkiːtoʊ/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

Java mocking framework whose name is a maintainer-confirmed pun on mock + mojito; a maintainer answered issue #889 with mock-based /mɑˈkiːtoʊ/ ("mah-KEE-toh"), while creator Szczepan Faber told the mailing list to "go ahead and pronounce it the mojito way" ("moh-KEE-toh"), so the split is genuine.

Alternate readings you might hear:

  • "moh kee toh" /moʊˈkiːtoʊ/

Source: mockito/mockito issue #889 — maintainer IPA answer
