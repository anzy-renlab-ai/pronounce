How to pronounce WSGI

acronym contested

WIZ gee /ˈwɪz ɡi/ mp3
or: double you S G I /ˌdʌbəl juː ɛs dʒiː ˈaɪ/

WSGI is most commonly pronounced "WIZ gee" (/ˈwɪz ɡi/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

"WIZ-gee" common; letter-by-letter also heard. Web Server Gateway Interface.

Alternate readings you might hear:

  • "double you S G I" /ˌdʌbəl juː ɛs dʒiː ˈaɪ/

Source: WSGI docs
