How to pronounce Citus

project creator-clarified

site us /ˈsaɪtəs/ mp3

Citus is most commonly pronounced "site us" (/ˈsaɪtəs/). This pronunciation is documented by the project itself (see Citus Data blog — "How do you pronounce Citus?" (pronounced "site-us")).

Citus is the open-source extension that turns PostgreSQL into a distributed, sharded database, and it powers Azure Cosmos DB for PostgreSQL. It is pronounced SITE-us, not KEE-tus or SIT-us — the name comes from the Olympic motto Citius, Altius, Fortius (faster, higher, stronger). Citus Data got asked so often that they published an official blog post settling it.

Source: Citus Data blog — "How do you pronounce Citus?" (pronounced "site-us")
