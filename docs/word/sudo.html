How to pronounce sudo

cli-tool contested

soo doo /ˈsuːduː/
or: soo doh /ˈsuːdoʊ/

sudo is most commonly pronounced "soo doo" (/ˈsuːduː/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

"soo-doo" (substitute-user-do, Bourne convention) vs "soo-doh".

Alternate readings you might hear:

  • "soo doh" /ˈsuːdoʊ/

Source: Wikipedia

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it sudo # primary × 3 + audible "or: …" for each alternate
$ say-it --alt sudo # focus on the first alternate
$ say-it --all sudo # primary AND every alternate, each repeated
$ say-it --solo sudo # primary only, no "or: …" tail
$ say-it --why sudo # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/say-it.git && cd say-it && ./install.sh
