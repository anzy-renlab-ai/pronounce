How to pronounce Arbitrum

product contested

AR bih trum /ˈɑːr bɪ trəm/ mp3
or: ar BIT rum /ɑːrˈbɪ trəm/

Arbitrum is most commonly pronounced "AR bih trum" (/ˈɑːr bɪ trəm/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

"AR-bih-trum" (Offchain Labs) common; "ar-BIT-rum" also heard.

Alternate readings you might hear:

  • "ar BIT rum" /ɑːrˈbɪ trəm/

Source: arbitrum.io

Pronouncing project and product names correctly avoids the small but persistent friction of being gently corrected during standups, conference Q&As, and team calls. Hearing the word a few times locks in the right reading better than reading IPA ever will. Pronounce is a community-maintained dictionary — every entry tagged with a confidence level and (where possible) a citable source.

Hear it from the command line

$ say-it Arbitrum # primary × 3 + audible "or: …" for each alternate
$ say-it --alt Arbitrum # focus on the first alternate
$ say-it --all Arbitrum # primary AND every alternate, each repeated
$ say-it --solo Arbitrum # primary only, no "or: …" tail
$ say-it --why Arbitrum # print the dict entry with source URL

Install the CLI: git clone https://github.com/anzy-renlab-ai/pronounce.git && cd pronounce && ./install.sh
