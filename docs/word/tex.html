How to pronounce TeX

tool creator-clarified

tek /tɛk/ mp3

TeX is most commonly pronounced "tek" (/tɛk/). This pronunciation is documented by the project itself (see Wikipedia § Pronunciation).

Knuth: like "tech" — the X is a Greek chi.

How do you pronounce TeX?

TeX is pronounced "tek" (/tɛk/). Knuth: like "tech" — the X is a Greek chi. Source: Wikipedia § Pronunciation.

What is the IPA for TeX?

The IPA for TeX is /tɛk/, respelled "tek".
