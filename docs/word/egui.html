How to pronounce egui

project creator-clarified

ee gooey /ˈiːɡuːi/ mp3

egui is most commonly pronounced "ee gooey" (/ˈiːɡuːi/). This pronunciation is documented by the project itself (see egui official README (emilk/egui)).

Rust immediate-mode GUI library by Emil Ernerfeldt (@emilk). README states verbatim: "pronounced as 'e-gooey'. Please don't write it as 'EGUI'." Lowercase, like "e" + the word "gooey". STRESS: EE-gooey (first syllable). Often mispronounced "ee-gwee" or spelled out E-G-U-I.

Source: egui official README (emilk/egui)
