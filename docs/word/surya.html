How to pronounce Surya

tool contested

soor yuh /ˈsuːrjə/ mp3
or: soo ree uh /ˈsuːriə/

Surya is most commonly pronounced "soor yuh" (/ˈsuːrjə/). This is one of several readings in active use; the developer community hasn't converged on a single answer.

Surya is Datalab's open-source OCR, layout and table-recognition model and Python package (surya-ocr), the engine behind Marker. The README says it is named for the Hindu sun god, Sanskrit/Hindi sources support SOOR-yuh (two syllables, long oo, y-glide), while English Wikipedia gives the anglicized three-syllable SOO-ree-uh; both are attested and no source establishes a winner. The common mistake is reading the first vowel like surrey.

Alternate readings you might hear:

  • "soo ree uh" /ˈsuːriə/

Source: Wikipedia — Surya (/ˈsuːriə/ SOO-ree-ə), the Hindu sun god the project is named after; Wiktionary Sanskrit सूर्य [ˈsuːɾjɐ]
