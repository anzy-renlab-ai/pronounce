How to pronounce APISIX

project creator-clarified

ay pee eye six /ˌeɪ piː ˈaɪ sɪks/ mp3

APISIX is most commonly pronounced "ay pee eye six" (/ˌeɪ piː ˈaɪ sɪks/). This pronunciation is documented by the project itself (see GitHub — apache/apisix issue #1190, maintainer @membphis: "'6' is a lucky number in China, so we would like to call it `API SIX`").

Apache APISIX is a top-level Apache Software Foundation cloud-native API gateway and AI gateway, a peer of Kong, Traefik and Envoy. Despite looking like one word, it is spoken as two tokens: the letters A-P-I followed by the number six. Maintainer @membphis confirmed this on the project tracker, explaining that 6 is a lucky number in China — hence the name.

Source: GitHub — apache/apisix issue #1190, maintainer @membphis: "'6' is a lucky number in China, so we would like to call it `API SIX`"
