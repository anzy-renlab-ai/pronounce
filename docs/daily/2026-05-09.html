Pronunciation of the day: mise

mise is pronounced "meez"  ·  /miːz/  ·  cli-tool

▶ meez mp3

Rust polyglot version manager + task runner by Jeff Dickey (jdx); the asdf successor. Name is short for the French culinary term "mise-en-place." Official docs: `mise` (pronounced "meez"), rhymes with "cheese." One syllable, voiced final z. Common wrong guesses: "my-z", "mice", "miz."

For the full source citation, alternate readings, and CLI usage, see the canonical page → mise.
