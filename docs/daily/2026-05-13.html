Pronunciation of the day: mentat

mentat is pronounced "men tat"  ·  /ˈmɛntæt/  ·  tool

▶ men tat mp3

AI coding tool by AbanteAI — currently MentatBot, a GitHub-native coding agent (earlier archived CLI shared the name). Name borrowed from the "Mentats" (human computers) of Frank Herbert's Dune, from Latin mentis / mentation. STRESS-pattern: MEN-tat (first syllable). "tat" rhymes with "cat"; "men" rhymes with "ten".

For the full source citation, alternate readings, and CLI usage, see the canonical page → mentat.
