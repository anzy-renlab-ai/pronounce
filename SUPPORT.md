# Support

Thanks for using **Pronounce**. Pick the channel that matches what you need.

## "How do I install / use it?"

Start with the [README quick-try](https://github.com/anzy-renlab-ai/pronounce#-try-it-in-30-seconds). It's two commands.

If you'd rather not install, the dictionary is on the web at **[pronounce.renlab.ai](https://pronounce.renlab.ai)** — same audio, same sources, no setup. Chinese landing at **[/zh.html](https://pronounce.renlab.ai/zh.html)**.

## "Is there an entry for X?"

Search the dictionary directly: <https://pronounce.renlab.ai/browse.html>. Hit `/` to focus the search bar, type the word.

If we don't cover it — see "I want to add a word" below.

## "I want to add a word"

Two paths:

1. **Open a PR** in 30 seconds — add a row to [`data/pronunciations.tsv`](data/pronunciations.tsv). See [CONTRIBUTING.md](CONTRIBUTING.md) for the column format. This is the canonical contribution path.
2. **Open an issue** using the [🔊 Add a pronunciation entry](https://github.com/anzy-renlab-ai/pronounce/issues/new?template=add-pronunciation.yml) template if you'd rather suggest than ship.

## "I want to upgrade a source citation"

Lots of entries currently sit at `community-consensus` because no single citable source was found at curation time. If you've spotted a creator quote, official FAQ, project Wikipedia § Pronunciation, podcast, or conference clip — open a PR or the [📜 Source upgrade](https://github.com/anzy-renlab-ai/pronounce/issues/new?template=source-upgrade.yml) issue.

## "Something's broken"

Open a [🐛 Bug report](https://github.com/anzy-renlab-ai/pronounce/issues/new?template=bug.yml). Include the surface (CLI / site / MCP / skill), what you did, what happened.

## "I have an idea"

Read [`IDEAS.md`](IDEAS.md) first — your idea might already be planned. If yes, drop a 👍 or comment. If no, open the [💡 Feature](https://github.com/anzy-renlab-ai/pronounce/issues/new?template=feature.yml) template.

## "I want to chat / vote on what's next"

[Discussions](https://github.com/anzy-renlab-ai/pronounce/discussions) is where wishlist / "what should we cover next" lives:

- [What words do you want next?](https://github.com/anzy-renlab-ai/pronounce/discussions/34) — vote here
- [Announcements](https://github.com/anzy-renlab-ai/pronounce/discussions/categories/announcements) — release notes

## Security

`SECURITY.md` covers responsible disclosure. Short version: please don't file public issues for vulnerabilities — see the policy.

## Response time

This is a community project run by volunteers. Expect a response within a few days for most threads. Dictionary PRs with a clean source citation usually merge same-day.
